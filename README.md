# 📅 Cool日历 (Cool Calendar)

Cool日历是一个基于 **Flutter** 框架开发的现代化日程管理应用。项目致力于提供流畅、高性能的用户体验，并集成了多项高级功能，如农历显示、网络日历订阅、以及借鉴原生 Android 库 **CalendarView** 实现的平滑视图切换动画。

## ✨ 核心亮点与特性

### 1. 现代化 UI/UX 设计

* **iOS 风格：** 采用极简的浅灰色背景和圆角白色卡片布局，提供干净、专业的视觉体验。
* **收缩动画：** 实现月视图与周视图之间的平滑**收缩/展开动画**，极大地优化了屏幕空间的利用效率和交互流畅度。
* **多颜色标记：** 在日历数字下方显示彩色圆点，根据日程的类型（自建/订阅）和自定义颜色进行直观区分。

### 2. 高级日程功能

* **数据持久化：** 使用 `sqflite` 数据库存储所有日程数据，确保数据安全。
* **网络订阅：** 支持通过 URL 订阅外部 `.ics` 格式的日历源（如：法定节假日）。
* **订阅源管理：** 专门的界面用于管理、删除已订阅的日历源，并允许用户为每个源添加**备注名称**。
* **数据备份：** 支持将日程数据导出为 JSON 文件进行备份，以及从 JSON 文件导入恢复。
* **日程提醒：** 集成本地通知功能，为定时日程提供准确的闹钟提醒。
* **农历支持：** 日历格子内集成农历日期和重要节假日显示。

## 🛠️ 技术栈概览 (Tech Stack)

| 领域 | 关键技术 / 依赖 |
| :--- | :--- |
| **框架** | Flutter (Dart) |
| **状态管理** | `Provider` |
| **本地存储** | `sqflite` (SQLite), `shared_preferences` (订阅源配置) |
| **日历组件** | `table_calendar` (结合 `AnimationController` 实现收缩动画) |
| **网络与解析** | `http`, `icalendar_parser` (ICS 格式解析), `lunar` (农历转换) |
| **UX 库** | `flutter_local_notifications`, `share_plus` |
